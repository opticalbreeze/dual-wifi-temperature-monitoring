_/
_/  Chromiumの自動更新を無効にする方法(Lubuntu)
_/
_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/

※aptによるupdateは有効のままとします。
  適宜 update-webdriver.sh でのアップデートを行ってください。

-------------------------------------------------
▼ Chromeのポリシーファイルを設定するための
   ディレクトリを作成
-------------------------------------------------
sudo mkdir -p /etc/opt/chrome/policies/managed


-------------------------------------------------
▼ 作成したディレクトリに以下のjsonを含む
   ポリシーファイルを追加
-------------------------------------------------
sudo nano /etc/opt/chrome/policies/managed/disable_autoupdate.json


+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
| disable_autoupdate.json
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
{
  "AutoUpdateCheckPeriodMinutes": 0,
  "ComponentUpdatesEnabled": false
}
